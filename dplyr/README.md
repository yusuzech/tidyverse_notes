1.Create(`muate()`) sum and percentage of sum for columns
=========================================================

Instead of doing:

``` r
mtcars %>%
    group_by(cyl) %>%
    transmute(mpg_sum = sum(mpg),
           disp_sum = sum(disp),
           mpg_pct = mpg/sum(mpg),
           disp_pct = disp/sum(disp)) %>%
    head() %>%
    knitr::kable()
```

Do:

``` r
mtcars %>%
    group_by(cyl) %>%
    transmute_at(vars(mpg,disp),
              funs(sum = sum(.),
                   pct = ./sum(.))) %>%
    head() %>%
    knitr::kable()
```
